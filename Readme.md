#  SPEED RACER - Android Car Game

**Developer:** Benano Neta

##  About the Project
This is an interactive 2D car game built natively in Android Studio using Kotlin. The player must dodge incoming obstacles and collect coins to achieve the highest distance possible. 

##  Key Features
* **Dual Control Modes:** The game can be played using traditional on-screen buttons or by physically tilting the device using the **Accelerometer Sensor**.
* **Difficulty Levels:** Choose between FAST and SLOW game modes.
* **Persistent High Scores:** Top 10 scores are saved locally using `SharedPreferences`.
* **GPS & Google Maps Integration:** Upon crashing, the app records the user's current GPS coordinates. In the High Scores menu, tapping a score updates the embedded Google Map Fragment to display the exact geographical location where the game ended.
